# Luyen-nghe-morse
Web luyện nghe morse cây nhà lá vườn
Có 3 mode:
  - nghe từng ký tự
  - nghe 1 từ
  - nghe 1 đoạn
có thể chỉnh tốc độ (WPM)
có thể chỉnh volume
web sẽ phát âm thanh và người dùng nhập đáp án vào ô trả lời
bấm "kiểm tra" hoặc phím ENTER để check
bấm "hiện đáp án" để hiện đáp án
check đáp án xong bấm "câu mới" để chuyển sang nội dung tiếp theo
